# Security Policy

## Supported Versions

Security fixes are developed on `main` until the project publishes versioned release lines.

## Reporting a Vulnerability

Please report suspected vulnerabilities through GitHub private vulnerability reporting:

https://github.com/varrek/ai_sdlc/security/advisories/new

If private reporting is unavailable, open a minimal public issue that does not include exploit details and ask for a private disclosure channel.

## Disclosure

We aim to acknowledge reports within 7 days and coordinate disclosure once a fix or mitigation is available. Please do not publicly disclose exploit details before maintainers have had a reasonable chance to respond.
